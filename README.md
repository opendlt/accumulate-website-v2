# Accumulate Website v2

Official website for the [Accumulate Network](https://accumulatenetwork.io) — delegated authority you can prove, across organizations.

## About

Accumulate is a governance-first blockchain designed for real-world identity and multi-party approval. This is the redesigned marketing and documentation website built with Next.js.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
